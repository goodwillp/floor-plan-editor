Lock/Unlock Functionality Test

Test Instructions

  1. Open the main application at http://localhost:5173
  2. Load a reference image using the image upload button in the toolbar
  3. Try to click and drag the image - it should be locked by default (no movement)
  4. Click the lock/unlock button (padlock icon) in the toolbar
  5. Try to click and drag the image again - it should now be movable
  6. Click the lock button again to lock it

Expected Behavior

  • Locked State: Image cannot be moved by clicking and dragging
  • Unlocked State: Image can be moved by clicking and dragging
  • Button State: Lock button should show locked icon when locked, unlocked icon when unlocked
  • Status Messages: Should show "Reference image locked" or "Reference image unlocked" in status bar
